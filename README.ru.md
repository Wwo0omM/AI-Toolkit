# AI Toolkit (Blender Add-on)

AI Toolkit — это аддон для Blender, который позволяет генерировать 3D модели и изображения с помощью AI через Hugging Face Spaces.

##  Возможности

-  Генерация 3D модели по изображению (Hunyuan3D)
	-  Автоматический импорт модели во viewport Blender
	-  Сохранение модели на рабочий стол
-  Генерация изображений по текстовому промту (FLUX)
	-  Настройка разрешения изображения
-  Поддержка разных API / моделей (частично)
-  Встроенный доступ к manual (документации)

##  Как это работает

Аддон подключается к Hugging Face Spaces через API:

- **Hunyuan3D** — генерирует 3D модель по входному изображению  
- **FLUX** — генерирует изображение по текстовому описанию  

*Для генерации изображений требуется Hugging Face API токен*

##  Установка

1. Скачайте аддон (.zip)
2. Откройте Blender
3. Перейдите в: `Edit → Preferences → Add-ons → Install`
4. Выберите `.zip` файл
5. Включите аддон

##  Требования

- Blender **4.0+**
- Интернет-соединение
- Hugging Face API токен (для генерации изображений)

##  Использование

### Генерация 3D модели

1. Добавьте входное изображение
2. Выберите модель (если доступно)
3. Нажмите `Generate`
4. Дождитесь генерации (~3–10 минут)

Результат:
- Модель автоматически появится в Blender
- Также сохранится на рабочий стол

---

### Генерация изображения

1. Введите текстовый prompt
2. Укажите разрешение
3. Вставьте Hugging Face API токен
4. Нажмите `Generate Image`

Результат:
- Изображение сохранится на рабочий стол

 Время генерации: ~1–2 минуты

##  Ограничения

- Генерация может занимать время 
- Некоторые функции находятся в разработке
- Генерация изображения по картинке (FLUX) — **не работает**
- Зависит от доступности Hugging Face Spaces
